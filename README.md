# trello-webhook-client

Basic Trello webhook client to dump Trello webhook requests.


## Deploying to Heroku

For this to run on Heroku, you need following environment variables to be set:

* `HOST` - `0.0.0.0`
* `TRUST_PROXY` - `true`
